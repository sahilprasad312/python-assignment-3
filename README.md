# Python Assignment 3

This repository contains Assignment 3 for Module 4: Functions and Modules in Python.

## Tasks

### Task 1: Calculate Factorial Using a Function

File: `Task1_Factorial.py`

This program:

- Defines a function named `factorial`
- Takes a number from the user
- Calculates the factorial using a loop
- Displays the final result

### Task 2: Using the Math Module for Calculations

File: `Task2_Math_Module.py`

This program:

- Takes a number from the user
- Uses the `math` module
- Calculates the square root
- Calculates the natural logarithm
- Calculates the sine value in radians

## How to Run

Run Task 1:

```bash
python Task1_Factorial.py
```

Run Task 2:

```bash
python Task2_Math_Module.py
```

If `python` does not work on Windows, use:

```bash
py Task1_Factorial.py
py Task2_Math_Module.py
```

## Requirements

- Python 3
- No external libraries are required
